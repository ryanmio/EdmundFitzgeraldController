SS Edmund Fitzgerald

Advanced 1:150 scale naval telemetry platform for autonomous marine research and systems validation.

Section 01 // Project Abstract

Autonomous Naval Systems Integration

This project represents a comprehensive systems engineering effort to convert a static 1:150 scale vessel into a fully instrumented naval telemetry platform. The platform serves as a testbed for custom ESP32-based RTOS firmware, real-time battery monitoring, and precision marine propulsion control systems.
