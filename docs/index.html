Mission Overview

This project is a high-fidelity tribute to the legendary Great Lakes vessel. It leverages a dual-core ESP32 architecture to handle real-time sensor data, FPV video streaming, and precision motor control via a custom TypeScript command center.

Technical Specifications

The core components and technologies powering the 1:150 scale telemetry platform.

Vessel Architecture

  • Scale 1:150 (1.5m)
  • Displacement 22 lbs
  • Construction 3D Printed / Modified Hull
  • Service Marine Grade Conversion

Compute & Logic

  • Processor ESP32 Dual-Core
  • Firmware Custom C++ / RTOS
  • Connectivity Dual-Band WiFi / Failover
  • Vision FPV Stream (ESP32-CAM)

Power & Propulsion

  • Main Battery 8.4V Li-Po
  • Logic Power UBEC 5V/3A Output
  • Drive Motor Brushed DC / ESC Controlled
  • Steering High-Torque Servo

Mobile Platform

  • Framework React Native / Expo
  • Language TypeScript
  • Protocol Custom UDP/TCP Telemetry
  • Interface Real-time Dashboard
